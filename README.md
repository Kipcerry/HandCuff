This script adds a complete handcuff and restraint system designed for realistic roleplay interactions.

Players can restrain others using handcuffs or tiewraps, drag restrained players, and place or remove them from vehicles. Each pair of handcuffs generates unique handcuff keys when used. These keys only work for the specific handcuffs they were created for, adding an extra layer of realism and security.

Handcuffs can be removed using the correct keys or, if enabled, by lockpicking. Tiewraps require scissors to be cut. When handcuff keys are used, the original handcuffs are returned to the player.

The script includes optional police-only restrictions and seamless integration with ox_inventory, with item images included.

Features

Handcuff and tiewrap restraint system

Unique handcuff keys per handcuff

Keys only work on their matching handcuffs

Handcuffs returned when unlocked with keys

Optional lockpicking for handcuffs

Scissors required to cut tiewraps

Drag restrained players

Put restrained players into vehicles

Remove restrained players from vehicles

Optional instant dragging after cuffing

Optional police-only usage for handcuffs and keys

Fully configurable via the Config file

Compatible with ox_inventory

Item images included

ox_inventory Setup (Required)

The following items must be added to ox_inventory/data/items.lua:

Handcuffs

Handcuff keys

Tiewraps

Scissors
